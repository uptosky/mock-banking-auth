## Getting Started

To get started developing on your local make sure you have `nodejs`, and install `typescript`:

```
npm install -g typescript
```

then run:

```
npm install
npm run watch
```

For development API runs here: `http://localhost:3000`

To run in a 'production' environment:

```
npm run build
npm run start
```
